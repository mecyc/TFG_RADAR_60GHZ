% Abstract en castellano
\renewcommand*\abstractname{Resumen}
\begin{abstract}
Con el desarrollo de la tecnología se han obtenido diferentes avances tecnológicos los cuales han beneficiado las actividades de la humanidad es por ello que en el siguiente proyecto se propone el desarrollo de una aplicación la cual opera mediante código Python el cual permite que por medio de la obtención de lecturas de diferentes de materiales sea posible el clasificarlos e indicarlos, definiendo el tipo de material, todo ello implementando el uso de un radar a una frecuencia de 60 GHz, con el cual es posible indicar cual es la probabilidad este presente el material en la lectura.

Dado que es necesario realizar un análisis de los datos obtenidos por medio de radar, se propone implementar técnicas de aprendizaje automático para poder reconocer los diferentes materiales que se estudiaran, es por ello por lo que se propone el algoritmo clasificador Random Forest en conjunto con la validación cruzada.

Para poder recopilar la información necesaria, será necesario hacer uso de la librería del radar que es proporcionada por el fabricante, esto con el objetivo de poder conseguir una serie de lecturas, las cuales deberán brindar la suficiente información para poder clasificar, indicar e identificar todos los objetos sometidos al estudio, se propone un análisis con un registro de treinta materiales considerando diez lecturas generadas de cada material.
\end{abstract}

\renewcommand*\abstractname{Descriptores}
\begin{abstract}
Aprendizaje Automático, Clasificador, Random Forest, Reconocimiento, Python, Sensor, Radar, Acconeer.
\end{abstract}

% Abstract en inglés
\renewcommand*\abstractname{Abstract}
\begin{abstract}
With the development of technology, different technological advances have been obtained which have benefited the activities of humanity, which is why in the following project the development of an application is proposed which operates through Python code which allows through the Obtaining readings of different materials, it is possible to classify and indicate them, defining the type of material, all this by implementing the use of a radar at a frequency of 60 GHz, with which it is possible to indicate what is the probability that the material is present in the reading.

Since it is necessary to carry out an analysis of the data obtained by means of radar, it is proposed to implement machine learning techniques to be able to recognize the different materials that will be studied, that is why the Random Forest classifier algorithm is proposed in conjunction with the cross validation.

In order to collect the necessary information, it will be necessary to make use of the radar library that is provided by the manufacturer, this in order to be able to obtain a series of readings, which must provide enough information to be able to classify, indicate and identify all the objects submitted to the study, an analysis is proposed with a record of thirty materials considering ten generated readings of each material.
\end{abstract}
